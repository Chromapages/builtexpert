# BuiltExpert Marketing Website

## Overview
This project is the official marketing website for **BuiltExpert**, a contractor growth brand under Chromapages. The website is the core sales asset for the brand. Its job is to explain what BuiltExpert does, who it helps, why it is different, and how qualified contractors can take the next step.

This is not just a brochure site. It is meant to support lead generation, sales qualification, trust-building, and future growth through service pages, audit offers, case studies, landing pages, and SEO content.

## Purpose
The website exists to help BuiltExpert:

- attract qualified contractors looking for better lead generation
- convert visitors into discovery calls and audit inquiries
- establish a clear contractor-focused market position
- communicate a simple and credible growth offer
- create a scalable foundation for future niche pages, offer pages, and campaigns

## Business Role
The website should function as:

- a **sales tool** that moves visitors toward a call or audit
- a **qualification layer** that filters out poor-fit leads
- a **trust asset** that shows proof, process, and clarity
- a **growth foundation** for future campaigns and niche expansion

## Core Positioning
BuiltExpert helps contractors get more qualified leads through websites, landing pages, local SEO, and contractor-focused growth systems.

Key differentiators:

- contractor-specific positioning
- built to generate qualified leads
- fast, clear, and conversion-focused pages
- straightforward process
- practical growth strategy instead of vague marketing talk
- ongoing support after launch

## Primary Goals
- Increase discovery call bookings
- Increase qualified audit inquiries
- Improve trust and perceived relevance with contractors
- Support outreach, referrals, SEO, and paid traffic
- Create reusable content and structural foundations for future growth

## Audience
Primary audience:
- growth-minded contractors and home service businesses that need more qualified leads and a stronger online presence

Secondary audience:
- specialty trades businesses and owner-operators ready to modernize their lead generation system

## Recommended Tech Stack
This project is intended to be built with:

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**

Optional supporting systems:
- headless CMS for structured content
- GA4 for analytics
- GTM for tag/event management
- scheduler integration
- form handling and spam protection

## Suggested Content Model
Structured content should support:

- services
- contractor niche pages
- case studies
- FAQs
- testimonials
- core page content
- audit offer pages
- future blog content

## Core Functional Requirements
- responsive across major breakpoints
- clear CTA paths
- contact and audit capture forms
- booking CTA integration
- editable content
- SEO-ready architecture
- analytics tracking
- staging and preview deployment support

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Build For Production
```bash
npm run build
```

### 4. Start Production Preview
```bash
npm run start
```

# SEO Strategy

## Purpose
This document defines the SEO strategy for the BuiltExpert marketing website, including keyword directions, metadata structure, heading hierarchy, internal linking logic, and schema markup recommendations.

---

## SEO Goals
- rank for contractor-focused service queries
- capture branded and non-branded demand
- support future trade and local page expansion
- improve click-through rates through stronger metadata

## Keyword Direction
### Branded
- BuiltExpert
- BuiltExpert contractor marketing
- BuiltExpert contractor websites

### Service Intent
- contractor website design
- landing pages for contractors
- local SEO for contractors
- contractor marketing audit
- HVAC website design
- roofing SEO
- plumber website design

## Page-Level Mapping
- `/` -> contractor marketing website / contractor lead generation
- `/services/contractor-websites` -> contractor website design
- `/services/local-seo` -> local SEO for contractors
- `/services/lead-system-audits` -> contractor marketing audit
- `/audit` -> contractor audit

## Heading Hierarchy Strategy
### Core Rules
- one clear H1 per page
- H2s define major sections
- H3s support subsections
- headings should be descriptive, not decorative

### Homepage Pattern
```txt
H1: Primary Value Proposition
H2: Services
H2: Audit
H2: Featured Work
H2: Process
H2: Why BuiltExpert
H2: FAQs
H2: Next Steps
```

### Service Page Pattern
```txt
H1: Service Name
H2: Who This Is For
H2: What’s Included
H2: Why It Matters
H2: How It Works
H2: FAQs
H2: Next Steps
```

## Schema Recommendations
- Organization
- WebSite
- Service
- FAQPage
- BreadcrumbList
- Article later
- LocalBusiness only if live strategy supports it

## Internal Linking
- homepage -> services
- homepage -> audit
- homepage -> work
- services -> service detail pages
- case studies -> relevant services
- audit -> contact or booking
- blog later -> service pages and audit page

## Recommendation
Focus on high-intent contractor-service visibility first, not vanity traffic.
